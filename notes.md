# Primitif value
- Tipe data dasar, bawaan bahasa.
- Tidak berupa object, hanya menyimpan nilai langsung.
- Disimpan di stack memory.
- Ada 8 jenis: 
    - byte, short, int, long → bilangan bulat
    - float, double → pecahan
    - char → karakter tunggal
    - boolean → true/false
